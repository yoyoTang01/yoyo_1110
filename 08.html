Switch 判斷式

Switch判斷是用來根據不同的條件執行不同的程式碼區塊，通常用於多重選擇的情況

        switch(表達式) {
            Case 值1：
                //當表達式等於值1時執行的程式碼
                break;
            Case 值2：
                //當表達式等於值2時執行的程式碼
                break;
            Case 值3：
                //當表達式等於值3時執行的程式碼
            default:
                //當表達式不符合任何情況時執行的程式碼
         }

迴圈

迴圈用來重複執行一段程式碼，直到特定條件不再符合為止

for迴圈

        for(初始化; 執行條件; 步進值) {
        //重複執行的程式碼
        }

for/in

針對陣列的項目做處理

        for(變數 in 陣列物件) {
            //重複執行的程式碼
        }

while迴圈

        while(執行的條件) {
            //重複執行的程式碼
        }

do while迴圈

無論如何至少會跑一次

        do { 
            //重複執行的程式碼
        }while (執行的條件);

brack 與 continue

break 語句可用於跳出迴圈，並會繼續執行該迴圈之後的代碼

continue 語句可以中斷迴圈中的本次代碼，如果出現了指定的條件，然後繼續迴圈中的下一次代碼。

        //break用來跳出迴圈
        for(let i = 0; i < 10; i++) {
            if(i == 5) {
                break; //當 i 等於 5 時跳出迴圈
            }
            //其他程式碼
        }

        // continue 用來跳過本次迴圈，進入下一次迴圈
        for (let i = 0; i < 10; i++) {
            if (1 % 2 == 0) {
                continue; //當 i 為偶數時，跳過本次迴圈
            }
            //其他程式碼
        }

type of

type of 運算子用來檢查變數或表達式的資料類型。

Boolean (布林)在中文里通常指计算机科学中的逻辑数据类型，其值只有兩種可能：真(True) 或 假(False)，用來表示判斷結果或控制程式流程，得名自數學家喬治·布爾(George Boole)。

布林值(Boolean Value)： 就是 True 或 False，例如 2 <= 3 的結果是 True，而 0 == 1 的結果是 False。
