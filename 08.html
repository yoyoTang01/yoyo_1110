Switch 判斷式

Switch判斷是用來根據不同的條件執行不同的程式碼區塊，通常用於多重選擇的情況

        switch(表達式) {
            Case 值1：
                //當表達式等於值1時執行的程式碼
                break;
            Case 值2：
                //當表達式等於值2時執行的程式碼
                break;
            Case 值3：
                //當表達式等於值3時執行的程式碼
            default:
                //當表達式不符合任何情況時執行的程式碼
         }

迴圈

迴圈用來重複執行一段程式碼，直到特定條件不再符合為止

for迴圈

        for(初始化計數器; 執行條件; 布進值) {
        //重複執行的程式碼
        }

for/in

針對陣列的項目作處理

        for(變數 in 陣列物件) {
            //重複執行的程式碼
        }
