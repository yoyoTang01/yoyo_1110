自訂函數

函數是用來封裝一段可以重複使用的程式碼區塊，可以用來執行特定的任務或計算

在Javascript中，可以使用function關鍵字來宣告一個函數

        //副程式
        function 函數名稱(參數) {
            //函數內的程式碼
            //不需要回傳
        }

判斷式

        
判斷式是用來根據條件執行不同程式碼區塊的結構，可以用來實現分支邏輯

        
IF判斷

        
            if(條件) {
                //條件為真時執行的程式碼
            } else {
                //條件為真時執行的程式碼
            }
        

        
多條件 IF 判斷

        
            if(條件1) {
                //條件1為真時執行的程式碼
            } else if(條件2){
                //條件2為真時執行的程式碼
            } else {
                ////以上條件皆不為真時執行的程式碼
            }
