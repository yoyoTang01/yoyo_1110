第1課 認識網站

了解網頁的組成

  • HTML
  • CSS
  • Javascript
  • Server Script
  • DataBase

HTML 文件主要藉由標記 (tags) 來標示文件中 語法的開始與結束。除了<br&lg、<hr&lg、<img&lg 等標記之外，大部分的標記都是成雙成對， 分別宣告該語法的開始與結束，在使用上並無大小寫之分。

CSS主要是為了加強網頁上的排版效果，它可用來定義 HTML 網頁上物件的大 小、顏色、位置與間距，甚至是為文字、圖片加 上陰影等等功能。

JavaScript 被設計為向 HTML 頁面增加交互性。使用 jQuery 函式庫可以極大地簡化了 JavaScript 編程。

PHP 是一種伺服端描述語言，用來製作動態網頁。它是開放原始碼 (Open Source) 而且是跨平台的。

所謂資料庫管理系統 (Database Management System, 簡稱為 DBMS)，它是透過一組程式模組來組織、管理、儲存和讀取資料庫的資料，任何使用者在操作資料庫時，都需要透過資料庫管理系統來處理。

軟體環境與安裝

  • 下載並安裝Visual Studio Code並安裝
  • 選擇「Use AI features with Copilot for free」並按下「Set up Copilot」
  • 登入GitHub啟用GitHub Copilot功能
  • 選擇Google Chrome開啟GitHub網站
  • 註冊GitHub，按continue授權
  • 回到VS CODE，將操作介面中文化，按Ctrl+Shift+P，輸入「lang」，接著選擇 「Configure Display Language」。
  • 將「locale」中的「en」改成「zh-TW」，儲存厚重開VSCODE即可
  • 介面左方點選左方的「Extensions」輸入「chinese」後點選「Chinese (Traditional) Language Pack」安裝擴充語系。安裝完畢後點選「Restart Now」重啟 VSCode。
